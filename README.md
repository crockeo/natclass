# natclass

Natural class tester (and hopefully solver).

### Setup

On a POSIX-compliant OS:

```bash
$ git clone https://github.com/crockeo/klasik
$ cd klasik
$ python3 loader.py # or ./loader.py

# From there, run main.py
$ python3 src/main.py # or ./src/main.py
```
